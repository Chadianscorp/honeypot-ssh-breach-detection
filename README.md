# Honeypot SSH Breach Detection

## Overview
This project documents a honeypot-based SSH breach detection exercise focused on identifying suspicious authentication activity, mapping observed behavior to relevant NIST controls, and translating findings into actionable security insights.

## What This Project Demonstrates
- Log review and threat detection.
- Suspicious SSH activity analysis.
- Control mapping to NIST guidance.
- Incident-style documentation and escalation thinking.

## Tools and Methods
- Honeypot environment.
- Event/log review.
- Firewall and access control review.
- NIST control mapping.

## Key Findings
- Repeated unauthorized SSH attempts were identified.
- Activity patterns suggested brute-force or probing behavior.
- Control mapping highlighted gaps and defensive opportunities.
